#  Enhanced Superstore Sales Analysis
# Overview

This project presents an in-depth Exploratory Data Analysis (EDA) of the Enhanced Superstore Sales Dataset. The objective is to uncover actionable business insights by analyzing sales, profit, discounts, customer behavior, and regional performance using Python data analysis libraries.

1. Objective

The main goals of this project are:

Perform data cleaning and preprocessing
Explore sales and profit trends across different dimensions
Analyze the impact of discounts on profitability
Identify top-performing products, categories, and regions
Derive meaningful business insights through visualization
2. Dataset Description

The dataset includes real-world retail transaction data such as:

Customer details (Name, ID, Segment)
Product information (Category, Product Name, Product ID)
Order details (Order Date, Order ID, Priority)
Sales metrics (Sales, Profit, Discount)
Geographic data (City, Country, Market)
🛠️ Tools & Libraries Used
Python 
Pandas  – Data manipulation
NumPy – Numerical computations
Matplotlib  – Data visualization
Seaborn  – Advanced visualization
Jupyter Notebook / Google Colab 

3. Data Preprocessing

The dataset was cleaned and prepared using the following steps:

Handled missing values using mean/median imputation
Removed duplicate records
Converted categorical variables into numeric format
Standardized and structured data for analysis
# Exploratory Data Analysis (EDA)

The following analyses were performed:

* Sales distribution across categories and sub-categories
* Profit trends by region and market
* Impact of discount on sales and profit
* Top-performing products and cities
* Customer segmentation insights
  
  5. Visualizations

A variety of visualizations were used to better understand the data:

Bar charts  – Category-wise performance
Pie charts  – Market share distribution
Histograms  – Sales and profit distribution
Scatter plots  – Discount vs Profit relationship
Heatmaps  – Correlation analysis
# Key Insights
Certain product categories contribute significantly to overall profit
Higher discounts do not always lead to higher profits
A small number of products generate a large portion of revenue
Regional performance varies significantly across markets
